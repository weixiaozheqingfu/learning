# 前奏

在servlet3.0规范之前，所有web项目都需要web.xml文件才可以正常启动，并且可以配置filter、servlet等信息。
在servlet3.0规范之后，web项目可以不需要web.xml文件也可以正常启动，配置filter、servlet等信息可以通过servlet3.0规范新增加的类和方法加以实现。
